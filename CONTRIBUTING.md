# Contributing to Core_Aim_pro

Thank you for your interest in Core_Aim_pro!

## Important Notice

This repository **does not contain source code**. Only compiled release binaries are distributed via [GitHub Releases](https://github.com/CougarP/Core_Aim_pro/releases). The source code is proprietary and not available for external contributions.

## How You Can Help

Even though code contributions are not accepted, there are several ways you can help improve Core_Aim_pro:

### Bug Reports

If you find a bug, please open an [Issue](https://github.com/CougarP/Core_Aim_pro/issues) with:

1. **Description** — What happened and what you expected
2. **Steps to reproduce** — How to trigger the bug
3. **Environment** — OS, GPU model, device (Titan Two / KMBox Net), console
4. **Logs or screenshots** — Any relevant evidence

### Feature Requests

Have an idea to improve accessibility or usability? Open an Issue with the tag `enhancement` and describe:

- What the feature would do
- Why it would help users with disabilities
- Any reference or example if applicable

### Testing and Feedback

- Test new releases and report issues
- Share your experience and accessibility needs
- Help other users in the Discord community

### Documentation

- Suggest improvements to setup guides
- Help translate documentation
- Report unclear or outdated instructions

## Community

Join the official Discord server for real-time support, discussions and announcements.

## Code of Conduct

Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md) in all interactions.
